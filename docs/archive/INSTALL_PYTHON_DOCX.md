# Установка python-docx для генерации PDF-актов

## Быстрая установка

Откройте командную строку (cmd) в папке с ботом и выполните:

```bash
pip install python-docx
```

## Полная установка всех зависимостей

Если хотите установить все зависимости сразу:

```bash
pip install -r requirements.txt
```

## Проверка установки

Проверьте, что библиотека установлена:

```bash
python -c "from docx import Document; print('python-docx установлен успешно!')"
```

Если команда выполнилась без ошибок и вывела сообщение об успехе - всё готово!

## Возможные проблемы

### Ошибка: "No module named 'docx'"

**Решение:** Установите библиотеку:
```bash
pip install python-docx
```

### Ошибка при конвертации в PDF

**Причина:** Библиотека `docx2pdf` требует Microsoft Word на Windows.

**Решение 1:** Установите Microsoft Word (если его нет)

**Решение 2:** Бот автоматически вернет DOCX-файл вместо PDF, если конвертация не удалась. Вы можете открыть DOCX в Word и сохранить как PDF вручную.

## Что делает python-docx?

Библиотека `python-docx` позволяет боту:
- Создавать документы Word (.docx) программно
- Заполнять шаблоны актов приема-передачи
- Добавлять таблицы с данными об оборудовании
- Форматировать текст и выравнивание

После создания DOCX, библиотека `docx2pdf` конвертирует его в PDF для удобства.
