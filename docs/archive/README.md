# Архив документации

Эта папка содержит старые документы, которые были созданы в процессе разработки проекта.

## Назначение

Документы в этой папке сохранены для:
- Истории разработки
- Справочной информации
- Технических деталей реализации
- Документации hotfix'ов и исправлений

## Актуальная документация

Для актуальной информации используйте основные документы:

- **[USER_GUIDE.md](../USER_GUIDE.md)** - Руководство пользователя
- **[FEATURES.md](../FEATURES.md)** - Описание функций
- **[TROUBLESHOOTING.md](../TROUBLESHOOTING.md)** - Решение проблем
- **[CHANGELOG.md](../CHANGELOG.md)** - История изменений

## Содержимое архива

### Hotfix документы
Документация исправлений и багфиксов

### Summary документы
Сводки по завершенным задачам и обновлениям

### Test документы
Руководства по тестированию конкретных функций

### Feature документы
Детальное описание отдельных функций

### Technical документы
Техническая документация и анализ проекта

---

**Примечание:** Документы в архиве могут быть устаревшими. Всегда используйте актуальную документацию из корня папки docs.
